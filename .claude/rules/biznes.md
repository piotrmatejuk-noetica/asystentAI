# BIZNES — PIOTR MATEJUK

---

## Działalność

Piotr prowadzi 5 projektów równolegle, każdy w innej niszy ale wszystkie wokół wspólnego świata — psychologia, świadomość, transformacja, kultura:

**SACRUM (sacrum.life)**
Centrum Terapii Psychodelicznych. Główny projekt. Hipnoterapia + terapia wspierana psychodelikami, praca indywidualna i grupowa. Model: usługi terapeutyczne, warsztaty, kursy.

**hipnoterapia.edu.pl**
Profesjonalna Szkoła Hipnoterapii. Platforma edukacyjna dla terapeutów i osób wchodzących w zawód. Model: edukacja, certyfikacja.

**egoisnt.com**
Streetwear dla psychonautów, rave lovers i miłośników psychologii. Drops-based, premium, self-ironic. Headless frontend (Next.js + Shopify), własny CRM w budowie.

**cienfestiwal.com**
Festiwal Cień — wydarzenie kulturalne.

**magdalenagajdzinska.pl**
Projekt żony — w planach pełen marketing hub (Google Ads, SEO, automatyzacje).

## Stack technologiczny

```

Auth:        NextAuth.js v5

Ads:         Meta Marketing API, Google Ads API (w planach)
Analytics:   Supermetrics, Vercel Analytics
Kreacje:     Canva
Deploy:      Vercel
AI:          Claude Code
```
